Weatherman homepage

Reports

Reports are used to condense the "Data dump" data so its easier to manage. When it is built out instead of being a list of weather reports like the data dump is it will be a list of storms. The storm data will have the start, stop, and any large fluxuations during it.

Data dump

Data dump returns a list of all weather data that matches the search perameters. You can select the ranges of weather type based on their sky_id. The list can be further refined if you have specific ranges of interest. Start and an end times are optional and highly customizable. If you add dates it needs a data at minimum. Hours, minutes, seconds are optional but data is polled in minutes so...

State

Poll data
